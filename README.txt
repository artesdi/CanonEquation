Программа приводит уравнение к каноническому виду: приводит подобные слагаемые и приравнивает сумму к нулю.

Уравнение может быть любого порядка, может содержать любое количество переменных, может быть записано со скобками.

На вход должны быть переданы строки в следующем формате:
P1 + P2 + ... = ... + PN
где P1..PN - слагаемые, представленные в виде:
ax^k
где a - число с плавающей точкой;
k - целое число; 
x - переменная (переменных у одного слагаемого может быть несколько).

Например, может быть дано уравнение следующего вида:
x^2 + 3.5xy + y = y^2 - xy + y
Которое будет приведено к виду:
x^2 - y^2 + 4.5xy = 0
